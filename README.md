# 1. Исследование SOTA и разработка веб-сервиса для удаления фона с фото (Python)

## 1.1. Цель работы

Цель работы — проанализировать современные методы (State‑of‑the‑Art), выбрать наиболее практичное 
решение и реализовать веб-сервис на Python, способный обрабатывать загружаемые пользователем изображения в 
реальном времени.

## 1.2. Исследование SOTA и выбор технологии

Ниже представлены наиболее заметные модели и библиотеки для удаления фона с изображений (данные на 2026 г.).

| Модель / Библиотека | Особенности | Качество (IoU/F1) | Скорость (FPS на GPU) | Сложность внедрения                               |
|----------------------|-------------|-------------------|--------|---------------------------------------------------|
| **U²‑Net** (2020) | Двухуровневая вложенная U‑архитектура; предобучена на DUTS | IoU ~0.93 | ~30–40 | Очень низкая (готовая библиотека)                 |
| **MODNet** (2021) | Оптимизирована для портретов; легковесна | IoU ~0.94 (на портретах) | ~60+ | Низкая (через `rembg` или официальный код)        |
| **BRIA RMBG‑1.4** | Фундаментальная модель, обучена на большом коммерческом датасете; выдаёт маску высокого разрешения | IoU ~0.96+ | ~15–25 | Средняя (есть Python‑пакет `briarmbg` или через `rembg`) |
| **SAM** (Segment Anything) от Meta | Универсальная модель, требует подсказки (точки/боксы) или используется в автоматическом режиме | Высокое, но не специализировано на фоне | Медленная без ускорителей | Высокая (вес ~2.4 ГБ)                             |
| **BiRefNet** (2024) | State‑of‑the‑art на бенчмарках DIS, высокая точность границ | IoU ~0.97 | ~10–20 | Средняя                                           |

<b>Критерии выбора для веб-сервиса</b>:
* Точность маски — способность корректно отделять сложные элементы (волосы, полупрозрачные объекты).  
* Скорость инференса — приемлемое время ответа.  
* Простота интеграции и развертывания — наличие готового Python‑пакета, автоматическая загрузка весов.  
* Универсальность — работа с любыми типами объектов, а не только с портретами.  

<b>Итоговый выбор</b>: библиотека [rembg](https://pypi.org/project/rembg/) с моделью `u2net` или более новой `bria-rmbg` (BRIA RMBG‑1.4).  
`rembg` предоставляет единый интерфейс, автозагрузку моделей и поддерживает как CPU, так и GPU (через ONNX Runtime). 
Для повышения точности в конфигурации используется модель `bria-rmbg`, которая даёт близкие к SOTA результаты при 
сохранении высокой скорости. Альтернативный вариант — использование `u2net` как облегчённой модели, если аппаратные 
ресурсы ограничены.  

# 2. Реализация веб-сервиса на Python (FastAPI + rembg)

## 2.1. Архитектура сервиса

* <b>Веб-фреймворк</b>: `FastAPI` (асинхронный, высокая производительность, автоматическая документация).  
* <b>Удаление фона</b>: `rembg` с моделью `bria-rmbg`.  
* <b>Обработка изображений</b>: `Pillow` и `io.BytesIO` для работы с байтовыми потоками.  
* <b>Формат ответа</b>: `PNG` с альфа-каналом (прозрачный фон) либо `JPEG` с белым фоном на выбор.  

## 2.2. Код веб-сервиса

Весь код представлен в файле [main.py](main.py)

## 2.3. Инструкция по использованию

Для скачивания репозитория и установки зависимостей необходимо выполнить следующие команды:  
```bash
git clone https://github.com/UgryumovAV/background_remover.git
cd background_remover
pip install -r requirements.txt
```

Запуск сервера:  
```bash
uvicorn main:app --host 0.0.0.0 --port 9952  # номер свободного порта.  
```

Использование:  
* При первом запуске необходимо дождаться скачивания модели.  
* Откройте браузер по адресу http://localhost:9952/docs – интерактивная документация Swagger.
* Выберите `endpoint` `POST /remove-bg`, загрузите файл (`JPG`/`PNG`).  
* Для получения `JPEG` с белым фоном включите флажок white_bg.  
* Ответ вернёт обработанное изображение.
